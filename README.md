# 🐱 土豆拾光

> 温暖记录每一个与宠物相伴的美好时光 💕

一个简洁优雅的宠物日常记录管理应用，帮助宠物主人轻松记录宠物的成长点滴。

---

## ✨ 功能特性

### 📱 宠物管理
- 添加宠物信息（名字、品种、性别、年龄、体重）
- 上传宠物头像
- 编辑/删除宠物

### 📝 八大记录模块
| 模块 | 功能 |
|------|------|
| 📔 日记 | 记录宠物日常趣事，支持图片上传 |
| 💉 疫苗 | 疫苗接种记录与到期提醒 |
| 🐛 驱虫 | 驱虫记录与周期管理 |
| 🛁 洗护 | 洗护记录与美容管理 |
| 🍖 饮食 | 饮食记录与营养管理 |
| 🏥 病例 | 病例记录与健康档案 |
| ✂️ 绝育 | 绝育手术记录 |
| ⚖️ 体重 | 体重追踪与趋势图表 |

### 📊 数据统计
- 体重趋势图表展示
- 月度消费统计
- 宠物成长报告

### 🔔 智能提醒
- 疫苗到期提醒
- 驱虫周期提醒
- 洗护周期提醒

### 💾 数据管理
- localStorage + IndexedDB 双存储方案
- 数据导出备份
- 数据导入恢复

## 🚀 快速开始

### 在线访问
```
https://zhikun-zhang.github.io/pet-memory/
```

### 本地运行
```bash
# 克隆项目
git clone https://github.com/zhikun-zhang/pet-memory.git

# 进入项目目录
cd pet-memory

# 使用浏览器打开
# 方式1：直接打开 index.html
# 方式2：使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```
