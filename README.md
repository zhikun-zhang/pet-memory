## 📝 项目简介

**土豆拾光** 是一款专为宠物主人设计的日常记录应用，帮助你记录宠物的成长点滴、健康状况和生活趣事。

- **产品定位**: 轻量级宠物记录工具
- **目标用户**: 宠物主人、宠物爱好者
- **核心价值**: 记录美好时光，守护宠物健康

---

## ✨ 功能特点

### 🐾 宠物管理
- 添加宠物基本信息（名字、性别、品种、生日等）
- 自定义宠物头像（支持图片压缩）
- 显示陪伴天数和体重趋势
- 点击卡片打开宠物详情页

### 📝 日记记录
- 支持添加文字描述和图片
- 图片存储采用 IndexedDB，避免 localStorage 存储限制
- 支持多种记录类型一键同步

### 📊 健康管理
- 💉 **疫苗记录**: 追踪疫苗接种情况
- 💊 **驱虫记录**: 提醒定期驱虫
- 🛁 **洗护记录**: 记录美容护理
- 🏥 **病例记录**: 管理健康状况
- ⚖️ **体重记录**: 图表展示体重变化趋势

### 💰 记账功能
- 自动汇总所有记录中的消费金额
- 按月份统计消费明细

### 📱 用户反馈
- 提交反馈（支持图片上传）
- 查看反馈记录
- 反馈查看器（可视化管理工具）

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 需要通过 HTTP 服务访问（支持 IndexedDB）

### 本地运行

```bash
# 方法一：使用 Python
python -m http.server 8000

# 方法二：使用 Node.js
npx serve .

# 方法三：使用 PHP
php -S localhost:8000
```

### 访问应用
打开浏览器访问 `http://localhost:8000/源代码/主应用.html`

## 💾 数据存储

### 存储策略
| 数据类型 | 存储方式 | 说明 |
|----------|----------|------|
| 宠物信息 | localStorage | 结构化数据，查询方便 |
| 记录数据 | localStorage | 结构化数据，查询方便 |
| 图片数据 | IndexedDB | 大容量数据，避免存储限制 |

### 数据备份
- **导出**: 设置 → 导出备份 → 保存 JSON 文件
- **导入**: 设置 → 导入恢复 → 选择 JSON 文件

## 📖 使用说明

### 添加宠物
1. 点击右上角 ⚙️ 设置按钮
2. 选择「添加宠物」
3. 填写宠物信息（名字、性别、品种等）
4. 点击保存

### 记录日记
1. 切换到「日记」标签页
2. 点击「+ 写日记」按钮
3. 选择宠物、填写内容
4. 可添加图片（最多4张）
5. 点击保存

### 查看统计
- **体重趋势**: 切换到「体重」标签页查看图表
- **消费统计**: 切换到「记账」标签页查看月度汇总
