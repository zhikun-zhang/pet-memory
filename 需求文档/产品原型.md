# 🎨 产品原型说明

## 📱 产品概述

**产品名称**: 土豆拾光  
**产品定位**: 宠物日常记录管理应用  
**目标用户**: 宠物主人  
**使用场景**: 记录宠物的日常、健康、饮食等信息

---

## 📋 功能模块

### 1. 宠物管理
- 添加宠物信息（名字、品种、性别、年龄、体重等）
- 上传宠物头像
- 编辑/删除宠物

### 2. 日记记录
- 记录宠物日常趣事
- 支持图片上传
- 记录时间自动生成

### 3. 健康管理
- 疫苗记录（接种时间、疫苗类型）
- 驱虫记录（驱虫时间、药物名称）
- 病例记录（症状、诊断、治疗方案）
- 绝育记录（手术时间、医院信息）

### 4. 生活记录
- 洗护记录（洗护时间、洗护项目）
- 饮食记录（食物种类、食量、时间）
- 体重记录（体重趋势图表）

### 5. 数据管理
- 数据导出备份
- 数据导入恢复
- 记账汇总展示

### 6. 用户反馈
- 反馈提交表单
- 反馈列表查看

---

## 🎯 设计原则

### 视觉设计
- **主题色**: 温暖橙色 (#ff9a6c)
- **背景色**: 米白色 (#fff8f0)
- **文字色**: 深棕色 (#5a3e36)

### 交互设计
- 点击宠物卡片打开详情页
- 下拉菜单操作状态变更
- 图片预览支持放大查看
- 平滑过渡动画效果

### 响应式设计
- 移动端优先
- 适配不同屏幕尺寸
- 触摸友好的按钮尺寸

---

## 📐 页面结构

### 首页
- 顶部导航栏（标题 + 宠物选择）
- 功能标签页（日记、疫苗、驱虫、洗护、饮食、病例、绝育、体重）
- 底部导航（首页、宠物、统计、设置）

### 宠物管理
- 宠物卡片列表
- 添加/编辑宠物表单
- 宠物详情页

### 设置页面
- 数据导出/导入
- 标签显示控制
- 用户反馈入口

---

## 🔄 数据流程

```
用户操作 → 数据验证 → 存储到 localStorage/IndexedDB → 界面更新
```

### 数据存储格式
- **宠物信息**: JSON 格式存储在 localStorage
- **记录数据**: JSON 格式存储在 localStorage  
- **图片数据**: Blob 格式存储在 IndexedDB

## 🎨 设计规范

### 颜色规范
| 颜色 | 值 | 用途 |
|------|-----|------|
| 主题色 | #ff9a6c | 按钮、强调元素 |
| 背景色 | #fff8f0 | 页面背景 |
| 卡片色 | #ffffff | 卡片背景 |
| 文字色 | #5a3e36 | 主要文字 |
| 辅助色 | #a08060 | 次要文字 |

### 字体规范
- 字体族: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- 标题: 18px, 加粗
- 正文: 14px, 常规
- 辅助文字: 12px, 常规

### 间距规范
- 卡片间距: 16px
- 内边距: 20px
- 圆角: 16px
